# 🌦️ Weather App

A simple and responsive Weather Application built using **Python**, **Flask**, **HTML**, **CSS**, and the **OpenWeatherMap API**. This application allows users to search for any city and view real-time weather information.

---

## 📌 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition
- 💧 Displays humidity
- 🌬️ Shows wind speed
- ⚡ Clean and responsive user interface
- 🔄 Real-time weather updates using API

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML5
- CSS3
- OpenWeatherMap API

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### 2. Go to the project folder

```bash
cd weather-app
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add your OpenWeatherMap API Key

Update your API key in `config.py` (or wherever your project stores it).

### 5. Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
